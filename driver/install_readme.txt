
winXP，控制面板，添加新硬件，从磁盘安装
WIN7， 打开设备管理器，打开”操作“菜单，然后“添加过时硬件”， 从磁盘安装，

WINXP, control pane -> add new hardware -> install from disk
WIN7, device manager -> operater -> Adding obsolete hardware -> install from disk
64 bit driver needs signature to install successfully.

注意：如果先前安装过旧版本驱动，需要先卸载，新版本程序不兼容旧驱动。
